Flashcards

JavaScript(JS)

An object-oriented computer programming language commonly used to create interactive effects within web browsers.

Operators

Constructs which behave generally like functions, but which differ syntactically or semantically from usual functions.

Variables

A value that can change, depending on conditions or on information passed to the program.

Variable naming conventions

When naming a class, the first word is lowercased while the second word will be capitalized.

Functions

A type of procedure or routine that performs a specific function.

Methods

Are the actions that can be performed on objects.

Arguments

Appear in procedure calls.

Parameters

Appear in procedure definitions.

Comments

A programmer-readable annotation in the source code of a computer program.

jQuery

A cross-platform JavaScript library designed to simplify the client-side scripting of HTML.

Attributes

A property or characteristic.

Return

a 'return' statement causes the program to leave the subroutine and return to the return address, directly after where the subroutine was called.

Chaining methods

Chaining allows us to run multiple jQuery methods, on the same element, within a single statement.

Strings

String is ANY text inside double or single quotes.

Booleans

The Boolean type is a data type, having two values (usually denoted True or False) intended to truth values of logic and Boolean algebra.

Undefined

The result of executing computer code that does not have a prescribed behavior by the language specification the code adheres to, for the current state of the program.

NaN

This property indicates that a value is not a legal number. It represents "Not-a-Number" value.

Escape

Makes a string portable, so it can be transmitted across any network to any computer that supports ASCII characters.

Alert

Something is wrong!
